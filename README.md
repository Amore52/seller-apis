# Интеграция с API Яндекс Маркета и Ozon
Этот проект предназначен для автоматизации обновления остатков и цен товаров на платформах Яндекс Маркет и Ozon. Он позволяет загружать данные о товарах, обновлять их остатки и цены с помощью API данных торговых площадок.

## Функциональность
* Получение списка товаров с Яндекс Маркета и Ozon.
* Обновление остатков товаров на обеих платформах.
* Обновление цен товаров, используя загруженные данные.
